# Physics-Informed Neural Networks (PINN)

Репозиторий содержит реализацию **физически-информированных нейронных сетей (Physics-Informed Neural Networks, PINN)** на базе **PyTorch** для решения дифференциальных уравнений.

PINN — это подход на стыке **машинного обучения и численных методов**, в котором физические законы (например, дифференциальные уравнения) включаются непосредственно в функцию потерь нейронной сети.  

Это позволяет обучать модель так, чтобы она **удовлетворяла физическим законам**, даже при отсутствии большого количества данных.

---

# Основные возможности

- реализация PINN на **PyTorch**
- использование **автоматического дифференцирования**
- поддержка:
  - начальных условий
  - граничных условий
  - физически-информированных ограничений
- модульная архитектура
- воспроизводимые эксперименты
- визуализация результатов

---

# Области применения

Physics-Informed Neural Networks могут использоваться для решения:

- уравнения теплопроводности
- уравнения Бюргерса
- волнового уравнения
- уравнения Пуассона
- задач гидродинамики
- обратных задач в физике

PINN минимизируют невязку дифференциального уравнения вместе с ошибкой данных.

# Быстрый старт

Запуск обучения:

```

python train.py

```

Типичный процесс обучения PINN:

1. Определяется дифференциальное уравнение
2. Генерируются точки коллокации
3. задаются граничные условия
4. обучается нейронная сеть
5. минимизируется невязка уравнения

---

# Пример

Пусть требуется найти решение уравнения:

```

u(x,t)

```

Нейронная сеть аппроксимирует решение:

```

u(x,t) ≈ NN(x,t; θ)

```

Функция потерь имеет вид:

```

Loss = L_data + L_boundary + L_physics

```

где

- **L_data** — ошибка по данным  
- **L_boundary** — ошибка граничных условий  
- **L_physics** — невязка дифференциального уравнения

# Обучение

Обучение PINN происходит следующим образом:

```

for epoch in range(EPOCHS):

sample_collocation_points()
compute_pde_residual()
compute_boundary_loss()
update_network_parameters()

```

Для вычисления производных используется **автоматическое дифференцирование PyTorch**.

# Теоретическая основа

Метод Physics-Informed Neural Networks был предложен:

**M. Raissi, P. Perdikaris, G. Karniadakis (2019)**

Идея состоит в том, чтобы включить **физические законы** непосредственно в процесс обучения нейронной сети.
